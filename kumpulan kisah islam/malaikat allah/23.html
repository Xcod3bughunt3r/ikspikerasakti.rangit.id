Malaikat Allah




Malaikat berbadan api dan salju


Malaikat berbadan api dan salju - Malaikat yang setengah badannya berupa api dan salju berukuran besar serta dikelilingi oleh sepasukan malaikat yang tidak pernah berhenti berzikir.

Ad-Dam'u, An-nuqmah, Ahlul Adli dan Malaikat bertubuh api dan salju - Tempat-tempat Bersejarah dalam Kehidupan Rasulullah oleh Hanafi Muhalawi Hal.22
